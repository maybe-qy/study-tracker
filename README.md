# 学业追踪与等效分报告 (Study Tracker)

面向高中学生（高一、高二、高三）的学业数据整合工具。追踪每次考试的成绩和排名，计算等效高考分，分析成绩趋势，生成 HTML 报告。

## 免责声明

1. 本 Skill 为第三方开源工具，非 Anthropics 官方开发，使用风险由使用者自行承担。
2. 技能包含本地 Python 脚本，建议执行前审阅脚本源码，避免敏感数据泄露。
3. 等效分计算仅供参考，不构成对高考成绩的预测。
4. 开发者不对因本工具造成的文件丢失、数据外泄、系统故障负责。

- "对比目标院校" / "帮我分析成绩"

## 功能

1. **初始设置** — 省份、高考年份、宏观数据导入、目标院校
2. **成绩录入** — 逐字段录入考试科目和排名
3. **等效分计算** — 3 种方法，按置信度权重加权融合，含误差区间
4. **趋势与波动分析** — 等效分序列追踪，浮动区间计算
5. **差距分析** — 对比目标院校录取线或给出分
