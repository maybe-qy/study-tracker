## 个体数据 Markdown 文件

路径：`data/personal/individual/[日期]_[考试名].md`

内容为每次考试的完整原始记录，格式：
```markdown
# [考试名]
- 日期：[日期]
- 类型：[考试类型]
- 年级：[年级]
- 语文：[分数]
- 数学：[分数]
...
```

写入后不可修改。如需更正，新建一份记录并标注"勘误"。
