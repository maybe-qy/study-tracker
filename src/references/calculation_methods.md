# 等效分计算方法 — 完整公式与边界规则

## 计算优先级

| 优先级 | 方法 | 置信度 | 触发条件 |
|--------|------|--------|---------|
| 1 | 双模块换算法 | A/B级 | `升级 Sheet 存在 AND 校内各科特控线/浙大线有数据` |
| 2 | 人数校准法 | B级 | `校内排名 != null AND 门槛数据（特控线上线人数）存在 AND 一分一段表存在` |
| 3 | 分数线对照法（等比例放缩） | A级 | `本次模考特控线 != null AND 上届高考特控线 != null` |
| 4 | 校排阈值估算法 | B级 | `升级 Sheet 存在 AND 语数英特控线+浙大线有数据` |
| 5 | 校内排名对照法 | A级 | `校内排名 != null AND 本校对照表_总分 Sheet 存在` |
| 6 | 单科排名对照法 | A级 | `选科校内排名 != null AND 单科对照 Sheet 存在` |
| 7 | 排名锚定法（交叉验证） | A级 | `(市排名或联盟排名) != null AND 一分一段表 Sheet 存在` |
| 8 | 校排名估算 | C级 | `校内排名 != null AND 学校类型 != null AND 本校对照表_总分 不存在` |

按优先级取第一个可用方法作为主方法，其余作为交叉验证。所有可用方法按置信度加权参与融合。

---

## 方法一：双模块换算法（优先级1，A/B级）

有校内各科划线数据时优先使用。将总分拆分为语数英（450分）和选科（300分）两个独立模块，各自用校内特控线/浙大线做比例换算。

### 模块一：语数英三科

```
语数英等效 = 高考语数英特控目标 + (高考语数英浙大目标 - 高考语数英特控目标)
           / (校浙大线 - 校特控线) × (学生语数英总分 - 校特控线)
```

高考参考目标（基于用户提供的 2025 浙江高考各科均衡目标的中点）：
- 特控线（592）→ 语数英 340（330~350）
- 浙大线（658）→ 语数英 378（374~383）

### 模块二：选科（每门满分100）

三级优先级：

| 优先级 | 条件 | 方法 | 置信度 |
|--------|------|------|--------|
| 1 | 校特控线 + 校浙大线都有 | `等效 = 90 + (96-90)/(校浙大-校特控) × (学生-校特控)` | A |
| 2 | 仅有校特控线 | `等效 = 90 × (学生/校特控)` | B |
| 3 | 无校内划线 | 赋分直映 → 跨次回退 → 粗略估算 | C/D |

高考参考目标：选科特控 90（88~92）、浙大 96（95~97）。

### 汇总

```
等效总分 = 语数英等效 + 物理等效 + 化学等效 + 技术等效
```

整体置信度：≥50%模块为A且无D → A；其余按多数原则降级。

## 方法三：分数线对照法/等比例放缩法（优先级3，A级）

### 公式

```
等效分 = (750 - 上届高考特控线) / (750 - 本次模考特控线) × (考生总分 - 本次模考特控线) + 上届高考特控线
```

### 计算示例

已知：
- 上届（2026）高考特控线 = 592
- 本次模考特控线 = 546.5
- 考生总分 = 650

```
等效分 = (750 - 592) / (750 - 546.5) × (650 - 546.5) + 592
       = 158 / 203.5 × 103.5 + 592
       ≈ 80.3 + 592
       ≈ 672.3
```

### 置信度说明

等比例放缩法和排名锚定法的数据来源本质同级——特控线和一分一段表都是省级官方数据，可靠性一致。两者的区别只在映射方式：排名锚定法通过排名百分位映射（中间无假设），等比例放缩法通过特控线线性放缩（假设难度线性对应）。线性假设不完全成立属于方法误差，而非数据来源不可靠。

### 边界条件
- 本次总分 ≤ 本次特控线：公式仍然有效（结果为负偏差），正常计算
- 本次总分 = 满分750：等效分 = 750
- 缺少任一特控线：跳过此方法

## 方法二：人数校准法（优先级2，B级）

利用校内门槛上线人数与高考一分一段表的人数映射关系，将校内排名转换为高考排名。



**示例**：
- 校内特控线上线人数：573人
- 高考特控线(592分)对应累计人数：64018人
- k = 64018/573 = 111.7
- 校内排第100名 → 高考排名 100×111.7 ≈ 11170名 → 查表得等效分 ~648分

**边界条件**：当校内排名 × k 超过一分一段表最大人数时，取最低分。

## 方法四：校排阈值估算法（优先级4，B级）（优先级3，B级）

利用校内两条已知线（特控线+浙大线）各自对应的校内排名，线性插值反推学生校内排名，再通过学校类型系数估算全市排名 → 一分一段表 → 等效分。

```
校排名 ≈ 浙大上线人数 + (特控上线人数 - 浙大上线人数) × (1 - (学生分-特控线)/(浙大线-特控线))
市排名 ≈ 校排名 / 全校总人数 × 一分一段表总人数 × 学校类型系数
等效分 = 一分一段表按百分位查分
```

学校类型系数：省重点 0.3, 市重点 0.6, 区重点 1.0, 普通 1.5。B级置信度。

## 方法五：校内排名对照法（优先级5，A级）（优先级4，A级）

### 步骤

1. 获取本次校内排名
2. 应用班型校准（如有）：
   - 未参加考试的重点班学生，排名计为"考前N名"
   - 校准后排名 = 本次排名 + 重点班未参考人数
3. 查询本校对照表_总分 Sheet，找到对应的往届高考总分
4. 如无对照表，降级为方法六

### 班型校准规则（一刀切）

未参加考试的重点班学生，排名计为"考前N名"，在估算全校排名时自动补在前面。

示例：
- 本次校内排名第80名（仅平行班参加）
- 重点班50人未参加
- 校准后排名 = 80 + 50 = 130名
- 查对照表：全校第130名 → 某高考分数

### 边界条件
- 对照表中无精确匹配排名 → 线性插值（最近两个排名之间）
- 排名超出对照表范围 → 取最近边界值，标注"对照表范围外"

## 方法六：单科排名对照法（优先级6，A级）

利用宏观数据中的单科对照Sheet（校内单科排名→高考等效分映射表），对每个选科独立查表。



**边界条件**：排名超出对照表范围时取边界值。

## 方法七：排名锚定法（优先级7，A级）/交叉验证（优先级5，A级）

### 公式

```
百分位 = 1 - (排名 / 总人数)
等效分 = 在一分一段表中查找相同百分位对应的分数
```

### 计算示例

已知：
- 全市排名：3000名
- 全市考生总数：约21000人
- 百分位 = 1 - (3000 / 21000) ≈ 0.857
- 在2026浙江一分一段表中，85.7%百分位对应约670分

### 边界条件
- 总人数是估算值 → 标注"用户估算"
- 一分一段表中无精确百分位 → 线性插值
- 总人数 < 排名 → 数据异常，返回 error

## 方法八：校排名估算（优先级8，C级）（优先级6，C级）

### 使用条件

仅校内排名、无本校对照表时使用。通过学校类型系数估算全市排名，再通过一分一段表锚定。

### 学校类型系数

| 学校类型 | 系数 | 说明 |
|---------|------|------|
| 省重点 | 0.3 | 学生整体排名靠前 |
| 市重点 | 0.6 | |
| 区重点 | 1.0 | |
| 普通 | 1.5 | |

### 公式

```
estimated_city_rank = (school_rank / school_total) × max_count × coeff
percentile = 1.0 - (estimated_city_rank / max_count)
```

### 边界条件
- estimated_city_rank 超过 max_count 时截断为 max_count
- percentile 限幅 [0, 1]
- 置信度 C 级（两次映射累积误差：校内→全市 + 全市→高考分）

## 多方法加权融合

所有可用方法（总分法 + 单科加总）按置信度权重加权平均。单科加总独立计算（语数英用分数线对照法，选科赋分直映），不与总分法恒等，确保融合产生有意义的交叉校验。

```
fused = Σ(score_i × weight_i) / Σ(weight_i)
```

每个方法按其置信度获得权重：

| 置信度 | 权重 |
|--------|------|
| A级 | 1.0 |
| B级 | 0.8 |
| C级 | 0.5 |
| D级 | 0.0（不参与） |

单科加总作为独立一项参与融合，权重 = avg(单科权重) × 0.5（衰减因子）。

**单科衰减因子 0.5**：单科等效分信息量低于总分法，衰减因子降低其在多方法融合中的比重。

**跨次回退折扣 0.85**：基于相邻考试间等效分相关性的保守估计（典型重测信度下限）。多步回退时累积衰减：discount = 0.85 ^ exams_skipped。

> 例：双模块 635.7(A,w=1.0) + 分数线 618.6(A,w=1.0) + 校排阈值 615.0(B,w=0.8) + 单科加总 599.2(w=0.45)
> fused = (635.7×1.0 + 618.6×1.0 + 615.0×0.8 + 599.2×0.45) / 3.25 ≈ 620.3

## 交叉验证

### 方法分歧处理（三档）

| 差异 | 处理 | 标注 |
|------|------|------|
| ≤3分 | 交叉验证一致 | "交叉验证一致，等效分可信度较高" |
| 3-5分 | 以主方法为准 | "方法间存在分歧（最大差异X分），以[主方法]为准" |
| >5分 | 建议补充数据 | "方法分歧较大（最大差异X分），建议补充排名或特控线数据以提高可靠性" |

## 误差区间计算

单一方法时，误差幅度基于置信度默认值：

| 置信度 | A | B | C | D |
|--------|---|---|---|---|
| 误差幅度 | ±5 | ±10 | ±15 | ±20 |

融合后，误差区间以融合值为中心，基于所有参与方法的分数与融合值的最大偏差扩展：
- 下界 = fused - max(margin, max_dev + 3)
- 上界 = fused + max(margin, max_dev + 3)
- max_dev = 所有方法分数与 fused 的最大偏差，"+3" 为安全边际
