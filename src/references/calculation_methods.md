# 等效分计算方法 — 完整公式与边界规则

## 计算优先级

| 优先级 | 方法 | 置信度 | 触发条件 |
|--------|------|--------|---------|
| 1 | 百分位排名锚定法 | A级 | `(市排名) != null AND 一分一段表 Sheet 存在` |
| 1 | 等比例放缩法（分数线对照法） | A级 | `本次模考特控线 != null AND 上届高考特控线 != null` |
| 2 | 校内排名对照法 | B级 | `校内排名 != null AND 本校对照表_总分 Sheet 存在` |

A 级方法按数据可用性触发，并列优先级 1。两者都可用时交叉验证，排名锚定法为主方法。

---

## 方法一：百分位排名锚定法（优先级1，A级）

### 公式

```
百分位 = 1 - (排名 / 总人数)
等效分 = 在一分一段表中查找相同百分位对应的分数
```

### 计算示例

已知：
- 全市排名：3000名
- 全市考生总数：约21000人
- 百分位 = 1 - (3000 / 21000) ≈ 0.857
- 在2026浙江一分一段表中，85.7%百分位对应约670分

### 边界条件
- 总人数是估算值 → 标注"用户估算"
- 一分一段表中无精确百分位 → 线性插值
- 总人数 < 排名 → 数据异常，返回 error

---

## 方法一（并列）：等比例放缩法/分数线对照法（优先级1，A级）

### 公式

```
等效分 = (750 - 上届高考特控线) / (750 - 本次模考特控线) × (考生总分 - 本次模考特控线) + 上届高考特控线
```

### 计算示例

已知：
- 上届（2026）高考特控线 = 592
- 本次模考特控线 = 546.5
- 考生总分 = 650

```
等效分 = (750 - 592) / (750 - 546.5) × (650 - 546.5) + 592
       = 158 / 203.5 × 103.5 + 592
       ≈ 80.3 + 592
       ≈ 672.3
```

### 置信度说明

等比例放缩法和排名锚定法的数据来源本质同级——特控线和一分一段表都是省级官方数据，可靠性一致。两者的区别只在映射方式：排名锚定法通过排名百分位映射（中间无假设），等比例放缩法通过特控线线性放缩（假设难度线性对应）。线性假设不完全成立属于方法误差，而非数据来源不可靠。

### 边界条件
- 本次总分 ≤ 本次特控线：公式仍然有效（结果为负偏差），正常计算
- 本次总分 = 满分750：等效分 = 750
- 缺少任一特控线：跳过此方法

## 方法二：校内排名对照法（优先级2，B级）

### 步骤

1. 获取本次校内排名
2. 应用班型校准（如有）：
   - 未参加考试的重点班学生，排名计为"考前N名"
   - 校准后排名 = 本次排名 + 重点班未参考人数
3. 查询本校对照表_总分 Sheet，找到对应的往届高考总分
4. 如无对照表，降级为方法三

### 班型校准规则（一刀切）

未参加考试的重点班学生，排名计为"考前N名"，在估算全校排名时自动补在前面。

示例：
- 本次校内排名第80名（仅平行班参加）
- 重点班50人未参加
- 校准后排名 = 80 + 50 = 130名
- 查对照表：全校第130名 → 某高考分数

### 边界条件
- 对照表中无精确匹配排名 → 线性插值（最近两个排名之间）
- 排名超出对照表范围 → 取最近边界值，标注"对照表范围外"

## 多方法加权融合

当多种方法均可用时，等效分按置信度权重加权融合，而非仅取优先级最高的方法：

```
加权等效分 = Σ(score_i × weight_i) / Σ(weight_i)
```

| 置信度 | 权重 |
|--------|------|
| A级 | 1.0 |
| B级 | 0.5 |
| C级 | 0（不参与） |

> 例：排名锚定法 670（A级，权重 1.0）+ 等比例放缩法 672（A级，权重 1.0）
> 加权等效分 = (670×1.0 + 672×1.0) / (1.0+1.0) = 671.0

## 交叉验证

### 方法分歧处理

| 差异 | 处理 | 标注 |
|------|------|------|
| ≤3分 | 加权融合 | "交叉验证一致，等效分可信度较高" |
| 3-5分 | 加权融合 | "方法间存在分歧（最大差异X分），已按置信度加权融合" |
| >5分 | 加权融合 | "方法分歧较大（最大差异X分），建议补充排名或特控线数据以提高可靠性" |

## 误差区间计算

多方法时，误差幅度由各方法分数的置信度加权标准差计算：
```
误差幅度 = max(3, 加权标准差 × 1.5)
误差区间下限 = 加权等效分 - 误差幅度
误差区间上限 = 加权等效分 + 误差幅度
```

无交叉验证（单一方法）时，默认误差幅度 = 5分（A级）/ 15分（B级）。
