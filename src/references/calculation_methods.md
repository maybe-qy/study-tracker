# 等效分计算方法 — 完整公式与边界规则

## 计算优先级

| 优先级 | 方法 | 置信度 | 触发条件 | 适用年级 |
|--------|------|--------|---------|---------|
| 1 | 百分位排名锚定法 | A/B级 | `(市排名) != null AND 一分一段表 Sheet 存在` | 高三(A) / 高二(B) |
| 2 | 校内排名对照法 | C级 | `校内排名 != null AND 本校对照表_总分 Sheet 存在` | 高三/高二 |
| 3 | 等比例放缩法（分数线对照法） | C级 | `本次模考特控线 != null AND 上届高考特控线 != null` | 高三/高二 |
| 4 | 校排名估算 | C级 | `校内排名 != null`，但无对照表，走升学率折算 | 高三/高二 |

高一不计算等效分，仅存档排名和分数。

---

## 方法一：百分位排名锚定法（优先级1，高三A级/高二B级）

### 公式

```
百分位 = 1 - (排名 / 总人数)
等效分 = 在一分一段表中查找相同百分位对应的分数
```

### 计算示例

已知：
- 全市排名：3000名
- 全市考生总数：约21000人
- 百分位 = 1 - (3000 / 21000) ≈ 0.857
- 在2026浙江一分一段表中，85.7%百分位对应约670分

### 边界条件
- 总人数是估算值 → 标注"用户估算"
- 一分一段表中无精确百分位 → 线性插值
- 总人数 < 排名 → 数据异常，返回 error

---

## 方法二：校内排名对照法（优先级2，C级）

### 步骤

1. 获取本次校内排名
2. 应用班型校准（如有）：
   - 未参加考试的重点班学生，排名计为"考前N名"
   - 校准后排名 = 本次排名 + 重点班未参考人数
3. 查询本校对照表_总分 Sheet，找到对应的往届高考总分
4. 如无对照表，降级为方法四

### 班型校准规则（一刀切）

未参加考试的重点班学生，排名计为"考前N名"，在估算全校排名时自动补在前面。

示例：
- 本次校内排名第80名（仅平行班参加）
- 重点班50人未参加
- 校准后排名 = 80 + 50 = 130名
- 查对照表：全校第130名 → 某高考分数

### 边界条件
- 对照表中无精确匹配排名 → 线性插值（最近两个排名之间）
- 排名超出对照表范围 → 取最近边界值，标注"对照表范围外"

---

## 方法三：等比例放缩法/分数线对照法（优先级3，C级）

### 公式

```
等效分 = (750 - 上届高考特控线) / (750 - 本次模考特控线) × (考生总分 - 本次模考特控线) + 上届高考特控线
```

### 计算示例

已知：
- 上届（2026）高考特控线 = 592
- 本次模考特控线 = 546.5
- 考生总分 = 650

```
等效分 = (750 - 592) / (750 - 546.5) × (650 - 546.5) + 592
       = 158 / 203.5 × 103.5 + 592
       ≈ 80.3 + 592
       ≈ 672.3
```

### 边界条件
- 本次总分 ≤ 本次特控线：公式仍然有效（结果为负偏差），正常计算
- 本次总分 = 满分750：等效分 = 750
- 缺少任一特控线：跳过此方法
- 无排名数据支撑，误差范围大于排名锚定法

---

## 方法四：校排名估算（优先级4，C级）

### 步骤

1. 用升学率或学校类型默认系数，将校排名折算为估算全市排名
2. 用估算出的排名位置，查一分一段表（如有）或本校对照表
3. 全程标注C级置信度

### 默认系数（学校类型）

| 学校类型 | 估算系数 |
|---------|---------|
| 省重点 | 校排名 × 15 ≈ 全市排名 |
| 市重点 | 校排名 × 8 ≈ 全市排名 |
| 普通高中 | 校排名 × 3 ≈ 全市排名 |

用户可手动调整系数。未明确学校类型时，默认使用市重点系数（保守估计）。

---

## 交叉验证

当多种方法均可用时：
- 主结果 = 优先级最高的方法
- 交叉验证 = 其他可用方法的结果
- 差异 ≤ 3分 → "等效分可信度较高"
- 差异 > 3分 → 标注差异值，用户自行判断

---

## 置信度在波动分析中的权重

| 等效分置信度 | 趋势提取权重 | 浮动区间权重 |
|------------|------------|------------|
| A级 | 1.0 | 1.0 |
| B级 | 0.8 | 0.8 |
| C级 | 0.5 | 0.5 |
| D级 | 不参与 | 不参与 |

---

## 误差区间计算

```
误差幅度 = max(3, |主结果 - 各交叉验证结果的平均值|)
误差区间下限 = 主结果 - 误差幅度
误差区间上限 = 主结果 + 误差幅度
```

无交叉验证时，默认误差幅度 = 5分（A级）/ 8分（B级）/ 15分（C级）。
