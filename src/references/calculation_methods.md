# 等效分计算方法 — 完整公式与边界规则

## 计算优先级

| 优先级 | 方法 | 置信度 | 触发条件 |
|--------|------|--------|---------|
| 1 | 分数线对照法（等比例放缩） | A级 | `本次模考特控线 != null AND 上届高考特控线 != null` |
| 2 | 校内排名对照法 | A级 | `校内排名 != null AND 本校对照表_总分 Sheet 存在` |
| 3 | 排名锚定法（交叉验证） | A级 | `(市排名或联盟排名) != null AND 一分一段表 Sheet 存在` |
| 4 | 校排名估算 | C级 | `校内排名 != null AND 学校类型 != null AND 本校对照表_总分 不存在` |

按优先级取第一个可用方法作为主方法，其余作为交叉验证。

---

## 方法三：排名锚定法/交叉验证（优先级3，A级）

### 公式

```
百分位 = 1 - (排名 / 总人数)
等效分 = 在一分一段表中查找相同百分位对应的分数
```

### 计算示例

已知：
- 全市排名：3000名
- 全市考生总数：约21000人
- 百分位 = 1 - (3000 / 21000) ≈ 0.857
- 在2026浙江一分一段表中，85.7%百分位对应约670分

### 边界条件
- 总人数是估算值 → 标注"用户估算"
- 一分一段表中无精确百分位 → 线性插值
- 总人数 < 排名 → 数据异常，返回 error

---

## 方法一：分数线对照法/等比例放缩法（优先级1，A级）

### 公式

```
等效分 = (750 - 上届高考特控线) / (750 - 本次模考特控线) × (考生总分 - 本次模考特控线) + 上届高考特控线
```

### 计算示例

已知：
- 上届（2026）高考特控线 = 592
- 本次模考特控线 = 546.5
- 考生总分 = 650

```
等效分 = (750 - 592) / (750 - 546.5) × (650 - 546.5) + 592
       = 158 / 203.5 × 103.5 + 592
       ≈ 80.3 + 592
       ≈ 672.3
```

### 置信度说明

等比例放缩法和排名锚定法的数据来源本质同级——特控线和一分一段表都是省级官方数据，可靠性一致。两者的区别只在映射方式：排名锚定法通过排名百分位映射（中间无假设），等比例放缩法通过特控线线性放缩（假设难度线性对应）。线性假设不完全成立属于方法误差，而非数据来源不可靠。

### 边界条件
- 本次总分 ≤ 本次特控线：公式仍然有效（结果为负偏差），正常计算
- 本次总分 = 满分750：等效分 = 750
- 缺少任一特控线：跳过此方法

## 方法二：校内排名对照法（优先级2，A级）

### 步骤

1. 获取本次校内排名
2. 应用班型校准（如有）：
   - 未参加考试的重点班学生，排名计为"考前N名"
   - 校准后排名 = 本次排名 + 重点班未参考人数
3. 查询本校对照表_总分 Sheet，找到对应的往届高考总分
4. 如无对照表，降级为方法三

### 班型校准规则（一刀切）

未参加考试的重点班学生，排名计为"考前N名"，在估算全校排名时自动补在前面。

示例：
- 本次校内排名第80名（仅平行班参加）
- 重点班50人未参加
- 校准后排名 = 80 + 50 = 130名
- 查对照表：全校第130名 → 某高考分数

### 边界条件
- 对照表中无精确匹配排名 → 线性插值（最近两个排名之间）
- 排名超出对照表范围 → 取最近边界值，标注"对照表范围外"

## 方法四：校排名估算（优先级4，C级）

### 使用条件

仅校内排名、无本校对照表时使用。通过学校类型系数估算全市排名，再通过一分一段表锚定。

### 学校类型系数

| 学校类型 | 系数 | 说明 |
|---------|------|------|
| 省重点 | 0.3 | 学生整体排名靠前 |
| 市重点 | 0.6 | |
| 区重点 | 1.0 | |
| 普通 | 1.5 | |

### 公式

```
estimated_city_rank = (school_rank / school_total) × max_count × coeff
percentile = 1.0 - (estimated_city_rank / max_count)
```

### 边界条件
- estimated_city_rank 超过 max_count 时截断为 max_count
- percentile 限幅 [0, 1]
- 置信度 C 级（两次映射累积误差：校内→全市 + 全市→高考分）

## 单科加总融合

总分等效分由**主方法分数**（总分法）与**单科加总法分数**按置信度加权融合得出，不对多个总分方法做加权融合。

```
fused = (primary_score × w_total + subject_sum × w_subject) / (w_total + w_subject)
w_total = avg(所有可用方法的权重)
w_subject = avg(单科权重) × 0.7
```

| 置信度 | 权重 |
|--------|------|
| A级 | 1.0 |
| B级 | 0.8 |
| C级 | 0.5 |
| D级 | 0.0（不参与） |

**单科衰减因子 0.7**：基于保守假设，单科等效分中包含的独立信息量约为总分法的70%。选科赋分的信息量天然低于总分法（赋分区间窄、与高考赋分不完全对应），语数英比例折算法依赖总分等效分的准确性，因此单科加总的整体权重需适当衰减。

**跨次回退折扣 0.85**：基于相邻考试间等效分相关性的保守估计（典型重测信度下限）。多步回退时累积衰减：discount = 0.85 ^ exams_skipped。

> 例：主方法分数 627（w_total = 1.0）+ 单科加总 633（w_subject = 0.56）
> fused = (627×1.0 + 633×0.56) / (1.0 + 0.56) ≈ 629.2

## 交叉验证

### 方法分歧处理（三档）

| 差异 | 处理 | 标注 |
|------|------|------|
| ≤3分 | 交叉验证一致 | "交叉验证一致，等效分可信度较高" |
| 3-5分 | 以主方法为准 | "方法间存在分歧（最大差异X分），以[主方法]为准" |
| >5分 | 建议补充数据 | "方法分歧较大（最大差异X分），建议补充排名或特控线数据以提高可靠性" |

## 误差区间计算

单一方法时，误差幅度基于置信度默认值：

| 置信度 | A | B | C | D |
|--------|---|---|---|---|
| 误差幅度 | ±5 | ±10 | ±15 | ±20 |

总分法与单科加总法融合后，误差区间以融合值为中心，两侧分别基于参与融合的方法偏差扩展：
- 下界扩展量 = max(margin, |fused - 主方法分数| + 3)
- 上界扩展量 = max(margin, |fused - 单科加总| + 3)
- 下限基于主方法偏差（更保守），上限基于单科偏差（波动更大），"+3" 为安全边际
